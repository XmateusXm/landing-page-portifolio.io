## 🚀 Projetos

### 📊 Análise de Dados com API do IBGE

Projeto desenvolvido para consumo e processamento de dados públicos utilizando Java com Spring Boot.

A aplicação consome uma API do IBGE para obter informações sobre estados e municípios, permitindo organizar e armazenar esses dados em banco de dados.

**Principais funcionalidades:**

* Consumo de API externa (IBGE)
* Processamento de dados de estados e municípios
* Persistência em banco de dados
* Consulta por UF (sigla do estado)

**Tecnologias:**
Java, Spring Boot, Spring Data JPA, MySQL

---

### ⚙️ API CRUD com Spring Boot

Aplicação backend focada na implementação de operações básicas de um sistema (CRUD).

O projeto foi desenvolvido para praticar a criação de APIs REST, organização em camadas e integração com banco de dados.

**Principais funcionalidades:**

* Criação, leitura, atualização e remoção de dados
* Estrutura em camadas (Controller, Service, Repository)
* Integração com banco de dados
* Endpoints REST

**Tecnologias:**
Java, Spring Boot, Spring Data JPA, MySQL
